## Build your data table

- Edit or remove caption
  To remove the table caption just delete all the text in the field marked `caption`
  To Edit type or paste required text in the field marked `caption`
- Edit add or remove table headers
  To add or remove table headers just edit the field marked `table headers`. \*Note each header row is an array item starting with `[` and ending with `]` (Open and closed quare brackets).
  Each header column is separated by `,` and structured as `{"headerText": "text","colSpan": number}` the text is fairly explanatory whilst the `colSpan` is the number of columns to span the table.
- Edit add or remove table rows and columns
  To edit the table body's rows and columns edit the field marked `table row text` as above each table row starts with the `[` and ends with the `]` (Open and closed quare brackets).
  Each columns text is wrapped in the `'` single quotes and separated by `,` comma.

## Copy your data-tables htnl and sanitize for Site-core

- Right click inside the table and click `inspect` for chrome `Inspect element` for firefox and IE.
- Traverse to the top of the `<table>` element.
- Click `Copy OuterHTML` this will save your table in your clipboard
- Goto online html tidy tool **[html cleaner](https://html-cleaner.com/)** as we will need to sanitize the code
- Paste your html in the right empty column
- ![remove only comments](/assets/images/clean-html.png)
- In the _cleaning options_ below make sure **remove comments** is the only selected option
- Paste code `from your clipboard` into the right-hand `mark-up` pane and click Clean HTML.
- Preview in the html editor on the left-hand `display` pane to check all looks ok.
- Copy and use the code in site-core HTML page editor.
